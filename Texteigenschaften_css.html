mit text-ident:__px; wird die Textzeile um __px eingerückt
z.B. p {
text-ident:30px;
}

mit text-align: right;
center;
justify; (Blocksatz) Ausrichtung

mit text-decoration:underline;
overline;
line-through; wird Text unter-, über-, durchgestrichen

mit letter-spacing:_px; wird die Laufweite vom Text eingestellt (Buchstabenabstand)

mit text-transform: capitalize; wird der 1. Buchstabe jeden Wortes groß geschrieben
uppercase; werden alle Buchstaben groß geschrieben
lowercase; werden alle Buchstaben klein geschrieben
