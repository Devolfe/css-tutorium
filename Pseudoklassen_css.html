Geile Sache bei der Zustandsanzeige von links



Link im Urzustand bekommt Farbe zugewiesen, in diesem Beispiel: blau

a:link {

color:blue;
}

a:visitet {
color:red;
} wurde er besucht, wird er rot angezeigt.

a:active {
background-color:yellow;
} verursacht die Gelbfärbung des Hintergrundes beim Klicken
